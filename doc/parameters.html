###Shape parameters

Variable Unit Description Influence Parameters name
Length(L) mm The total target body length Length of the target …shape$L
\(\rho_c\)/L ratio Ratio of the radius of curvature (\(\rho_c\)) and L Curvature of the bent cylinder, if no shape profile is loaded ( 8 for straight cylinder) …shape$rho_L
L/a ratio L/a with a the radius of the mid-point of the cylinder is the ratio of L to a Contains information about the width of the target. …shape$L_a
Taper order float The taper order (n) controls the tapering. For a straight cylinder (\(\rho_c\)/L -> infinity), n=2 results in a prolate spheroid if L>2a, or in an oblate spheroid for L<2a …shape$order
Length average boolean The average model over a range of lengths will be produced 1 = perform average; 0 = No average
std(L)/L mm The ratio of the standard deviation of length (std(L)) to the mean length (L) Defines the variability of the target lengths, used to compute the input models for the averaged model output …shape$Lstd
Length increment mm This value has to be chosen with care, as it has to be possible to calculate the provided std(L)/mean(L) around the provided mean L with the given increment. Defines the length increment of the target lengths, used to compute the input models for the averaged model output …shape$dL
Shape profile filepath Path to a profile file containing 3 or 5 columns. (X, Z, R coordinates and optional Xupper Xlower). The profile file must contain at least 3 columns, containing the center coordinates (X, Z, R) of each circular segment Defines the shape of the target, if no profile is selected a uniformly bent cylinder will be computed …shape$prof_name
Taper smooth float Filter length to smooth the radius (R) 1 = no smoothing, otherwise smoothes the tapering (reduces complexity) …shape$taper_sm
Axis smooth float Filter length to smooth the body axis (X, Z) 1 = no smoothing, otherwise smoothens the axis (flatter) …shape$axis_sm
